## рҹҡҖ Installation

### Method 1 вҖ” One-Line Install (Recommended)

```bash
git clone https://github.com/SHalimoosavi/syj-one.git && cd syj-one && bash install.sh
```

### Method 2 вҖ” Step by Step

**Step 1: Update Termux & install Git**
```bash
pkg update && pkg upgrade -y
pkg install git python -y
```

**Step 2: Clone the repository**
```bash
git clone https://github.com/SHalimoosavi/syj-one.git
cd syj-one
```

**Step 3: Run the installer**
```bash
bash install.sh
```

**Step 4: Restart Termux**
```bash
exit
# Open Termux again, then:
syj
```

**Step 5: First-time setup (add your API keys)**
```bash
syj config setup
```

> рҹ’Ў **That's it.** SYJ ONE is installed at `~/.syj-one/` and `syj` is available globally.

---

### Method 3 вҖ” Manual (Without install.sh)

```bash
# Install Python dependencies
pip install rich requests dnspython anthropic pypdf reportlab \
            python-dotenv tabulate schedule

# Make launcher executable
chmod +x syj

# Add to PATH (add this to your ~/.bashrc)
export PATH="$PATH:$(pwd)"

# Run
python3 syj_main.py
```

## рҹ”‘ API Keys Setup

SYJ ONE works **without any API keys** for most modules. Keys unlock premium features:

| Key | Module | Required For | Get It |
|---|---|---|---|
| **Anthropic** | `syj ai` | All AI features | [console.anthropic.com](https://console.anthropic.com) |
| **GitHub Token** | `syj dev` | Create repos, manage issues | [github.com/settings/tokens](https://github.com/settings/tokens) |

```bash
# Add keys via CLI
syj config set api_keys.anthropic sk-ant-your-key-here
syj config set api_keys.github ghp_your-token-here

# Or run the interactive wizard
syj config setup

# Verify
syj config show
```

> рҹ”’ Keys are stored locally at `~/.syj-one/config/settings.json` вҖ” never transmitted anywhere except the respective APIs.
